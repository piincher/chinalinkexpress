## LINK BUILDING BEST PRACTICES

### White Hat Only

**DO:**
- Create valuable, link-worthy content
- Build genuine relationships
- Provide expert value to journalists
- Guest post on relevant sites
- Get listed in legitimate directories
- Earn links through original research

**DON'T:**
- Buy links (penalty risk)
- Use link farms
- Participate in link exchanges
- Use automated link building
- Spam comments with links
- Create fake profiles

### Anchor Text Strategy

| Type | Percentage | Examples |
|------|------------|----------|
| Branded | 40% | "ChinaLink Express", "ChinaLink" |
| Naked URL | 20% | "chinalinkexpress.com" |
| Partial Match | 25% | "logistics company China Mali", "freight forwarder Bamako" |
| Exact Match | 10% | "cargo chine mali", "import chine mali" |
| Generic | 5% | "click here", "learn more" |

### Link Velocity

**Target Growth:**
- Months 1-3: 5-10 links/month (foundation)
- Months 4-6: 10-15 links/month (acceleration)
- Months 7-12: 15-20 links/month (maturity)

**Warning Signs:**
- Sudden spike in links (penalty risk)
- Too many exact-match anchors
- Links from irrelevant sites
- Links from penalized domains
